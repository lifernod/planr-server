> Planr - сервис для мгновенной организации совместных встреч
> и мероприятий без лишних звонков и переписок. Позволяет
> создавать события, автоматически приглашать участников
> через популярные мессенджеры (Телеграм, Вконтакте), а так
> же напоминать о встречах до их начала!

 
## Актуальность

Существует множество различных платформ связи, через которые каждому потенциальному участнику приходится отправлять какое-либо сообщение с приглашением на мероприятие или с информацией о нем.

## Описание продукта
Проект представляет собой цифровую платформу, призванную оптимизировать процесс организации совместных мероприятий (встреч, учебных собраний, походов в кино, игровых сессий). 
Основная идея заключается в переходе от хаотичного согласования в мессенджерах к структурированному и автоматизированному процессу.
Продукт состоит из трех компонентов: основной вебсайт, Telegram-бот (с Web App) и бот в ВК (с Mini App). 
Центральным элементом является механика “событий” и “приглашений”, которая позволяет создателям быстро собирать людей, а участникам – получать релевантные приглашения и напоминания.


Продукт позволяет:
- Создавать совместные события, добавлять им определенные атрибуты и рассылать приглашения на данное событие
- Просматривать участников события, их пожелания или причины отказа от участия
- Получать приглашения на совместные события
- Получать напоминания о предстоящих событиях

3. Различные объединения (кружки, команды, творческие объединения, проводящие мероприятия и т.п.)

## Предполагаемый функционал
Продукт планируется разбить на две версии:
- Основной функционал (MVP)
- Сторонние сервисы (ТПУ Авторизация, ТПУ Расписание, OpenStreetMap)

_Версия 1. Основной функционал_

- Авторизация (через телеграмм/вк ботов/миниаппов)
- Привязка профиля телеграмм/вк к сервису
- Создание события, присваивание атрибутов (организатор, место, дата, время проведения, предполагаемая длительность мероприятия)
- Приглашение участников на событие (вручную с помощью уникальной ссылки, автоматически с помощью рассылки для Группы)
- Просмотр недавних контактов
- Отправка уведомлений в привязанные сервисы (телеграм/вк)

_Версия 2. Сторонние сервисы_

- Авторизация с помощью Единого портала авторизации ТПУ
- Автоматическое определение оптимального времени проведения события исходя из учебного расписания всех участников группы (с помощью Расписания ТПУ)
- Возможность выбирать место проведения на карте города (с помощью OpenStreetMap)
